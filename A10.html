Cuadro Comparativo de Lenguajes Web

Lenguaje ¿Para qué sirve? Ventajas Desventajas
HTML Se usa para crear la estructura de las páginas web Es fácil de aprender No es un lenguaje de programación
CSS Sirve para dar diseño y estilo a las páginas Hace que las páginas se vean más bonitas A veces es difícil acomodar los elementos
JavaScript Permite agregar interactividad a la página Hace las páginas más dinámicas Puede ser complicado al inicio
